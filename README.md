# Cars Service

- rejestracja użytkowników przez email i hasło
- dostęp do aplikacji tylko dla zarejestrowanych użytkowników
- weryfikacja email
- dodawanie, edycja i usuwanie samochodów
- wyszukiwarka samochodów po różnych kolumnach
